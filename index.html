Team 9: Real-Time Simulation of a Microgrid

Senior Design Project (2024-2025)

Project Overview

This website is created to showcase our senior design project for 2024-2025. The project focuses on the real-time simulation of a microgrid integrating distributed energy resources (DERs) to ensure grid resilience and stability. Our work explores renewable energy integration, decentralized control systems, and real-time simulations.

Project Objectives

  • Develop a real-time simulation environment for a microgrid using MATLAB Simulink and RT-LAB.
  • Integrate distributed energy resources (DERs) like solar and wind energy into the microgrid simulation.
  • Explore decentralized control systems to maintain stability during varying grid conditions.
  • Simulate fault-tolerance and grid resilience under different operational scenarios.
  • Implement the project using Raspberry Pi and real-time controllers for testing and validation.

Project Timeline & Milestones

  • September 2024: Initial research and design proposal submitted.
  • October 2024: Simulation environment set up and baseline tests completed.
  • November 2024: Integration of DER models and real-time simulation framework.
  • December 2024: Mid-year review and feedback from National Grid stakeholders.
  • January 2025: Full implementation and fault-tolerance simulations.
  • March 2025: Testing on hardware using Raspberry Pi.
  • April 2025: Final presentation and project delivery.

Technical Details

Our project employs the following key tools and technologies:

  • MATLAB & Simulink: For real-time simulation of microgrid dynamics.
  • RT-LAB: To ensure real-time control and fault-tolerance testing.
  • Raspberry Pi: Used for decentralized control implementation.
  • Distributed Energy Resources (DERs): Solar and wind energy integration into the grid.
